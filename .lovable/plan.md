

# إضافة حسابات تجريبية وتفعيل الدخول

## ما سيحدث:

1. **تفعيل تأكيد الإيميل التلقائي** — حتى تعمل الحسابات مباشرة بدون رسالة تأكيد

2. **إنشاء edge function لتوليد الحسابات** — دالة `seed-users` تنشئ:
   - **قائدة**: `leader@sana.test` / كلمة السر: `Test1234`
   - **مدخلة بيانات**: `dataentry@sana.test` / كلمة السر: `Test1234`
   - مع تعيين الأدوار تلقائياً وربطهم بالمسارات

3. **تعبئة جدول صفحات القرآن** — إضافة الـ 1,211 صف عبر migration لتفعيل حساب الأوجه

4. **استدعاء الدالة** لإنشاء الحسابات فوراً

## بعد التنفيذ:
- تدخلين بإيميل `leader@sana.test` وكلمة سر `Test1234` كقائدة
- أو `dataentry@sana.test` كمدخلة بيانات
